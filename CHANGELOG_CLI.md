# CLI Changelog

## 1.0.0 - CLI Engine Upgrade

- Introduced a structured command system with help, list, and aliases.
- Added argument and option parsing with validation and suggestions.
- Added output formatting with log levels, tables, and progress indicators.
- Implemented diagnostics (`about`, `doctor`) and cache clearing.
- Added generators (`make:*`) with stubs, dry-run, and force flags.
- Added database tooling with migrations and seeders.
- Added development utilities (`serve`, `logs:tail`, completions).
- Preserved legacy commands for backward compatibility.
